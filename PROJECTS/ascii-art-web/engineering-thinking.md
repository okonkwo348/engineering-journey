# Engineering Thinking Challenge

Topic:
Responsive Design

Question

Suppose your company asks you to support four devices:

Phone

Tablet

Laptop

Ultra-wide Monitor

How would you design your CSS so that the layout looks good on all four without writing four completely different websites?

Hints

Think about:

- width

- max-width

- min-width

- media queries

- percentage widths

Goal

Explain your approach like an engineer rather than writing code.
